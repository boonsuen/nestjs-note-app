# NestJS Note App

- Technologies
  - NestJS
  - REST API
  - Postgres database
  - TypeORM
- Live website: <a href="https://nestjs-note-app.altisjs.com/">https://nestjs-note-app.altisjs.com/</a>
- API endpoint: <a href="https://api.nestjs-note-app.altisjs.com/">https://api.nestjs-note-app.altisjs.com/</a>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
